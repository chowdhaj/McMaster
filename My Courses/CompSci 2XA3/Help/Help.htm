1. Lab safety manual -- every student must read it, and there will be a quiz on the basics of
    safety included in the first lab. Without submitting the answers to the safey quizz, your
    projects will not be marked.
     
  2. If you are on Mac campus and using the wireless network to connect to moore, you must
    use MacSecure network.

  3. Change your moore password at https://www.cas.mcmaster.ca/account/
     
  4. If you have a Mac computer or a LINUX based computer, it has a native terminal. Run
    the terminal and use ssh moore.mcmaster.ca to connect to the server.
     
  5. If you have a Windows based computer, download and install a terminal emulator Ssh
    Secure Shell, see below.
     
  6. Download the installer ssh-3.2.exe for Ssh Secure Shell terminal emulator for Windows.
    Execute the installer ssh-3.2.exe and it will install Ssh Secure Shell emulator software on
    your Windows machine. See a brief description of how to connect to a host machine
    using Ssh Secure Shell     . See a brief description of how to transfer files to/from a host
    machine using Ssh Secure Shell
