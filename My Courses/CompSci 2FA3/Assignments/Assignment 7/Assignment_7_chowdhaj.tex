    uppercase letters in the Roman alphabet plus the hyphen symbol
    ``-''.  Let a \emph{simple English word} be a nonempty string $x
    \in \Sigma^*$ such that:

    \be

      \item An uppercase letter can only occur as the first symbol in
        $x$.

      \item A hyphen cannot occur as either the first or the last
        symbol in $x$.

      \item Two hyphens cannot occur in a row.

    \ee

    Write a regular expression that matches the set of simple English
    words (as defined above).
